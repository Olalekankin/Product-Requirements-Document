# Engagement Principles

These principles define how The Remote Jobs Connector grows and sustains its audience.

Source: knowledge-base.json v1.1.0 — contentObjectives + ctaLibrary

---

## Content Objectives

**Goals of every post:**
- Encourage follows
- Encourage comments
- Encourage reposts
- Build credibility
- Make people return for more job opportunities

**Guiding rule:** The call-to-action should never feel forced. Every CTA should read like a natural next line, not a tactic bolted onto the end of the post.
